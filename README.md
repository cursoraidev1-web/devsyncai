# Zyndrx Platform

A comprehensive project management and development coordination platform that brings together product managers, designers, developers, testers, and DevOps engineers in one collaborative space.

## Features

- **PRD Designer**: Collaborative workspace for product requirements
- **Task & Feature Tracker**: Kanban-style board for project management
- **Documentation Store**: Centralized repository for all project files
- **CI/CD Integration**: Track commits, builds, and deployments
- **Role Dashboards**: Custom views for each team role
- **Analytics & Reporting**: Performance insights and metrics
- **Team Collaboration**: Comments, mentions, and notifications

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Building for Production

```bash
npm run build
```

## Project Structure

```
src/
  ├── components/       # Reusable UI components
  ├── pages/           # Main page components
  ├── styles/          # CSS files
  ├── App.js           # Main app component
  └── index.js         # Entry point
```

## Technology Stack

- React 18
- React Router for navigation
- Modern CSS for styling
- No TypeScript (JavaScript only)
