## Overview

Zyndrx is a comprehensive platform that brings together product managers, designers, developers, testers, and DevOps engineers in one collaborative space.

## Features

- **PRD Designer**: Collaborative workspace for product requirements
- **Documentation Store**: Centralized repository for all project files
- **Task Tracker**: Kanban-style board for managing work
- **Role Dashboards**: Custom views for PM, Developer, QA, and DevOps
- **CI/CD Integration**: Track commits, builds, and deployments
- **Analytics & Reporting**: Monitor project metrics and team performance
- **Notifications**: Automated handoffs and alerts

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
```

- **Product Manager/Owner**: Manage PRDs, track progress, approve changes
- **Developer**: View assigned tasks, track code status
- **QA Team**: Manage test cases and issues
- **DevOps**: Monitor deployments and build logs
- **Admin**: Full system access and user management
