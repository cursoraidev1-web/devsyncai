## Getting Started

### Creating Your Account

1. **Sign Up**:
   - Visit [app.codexflow.io](https://app.codexflow.io)
   - Click "Sign Up" and enter your email address
   - Create a strong password (minimum 8 characters)
   - Verify your email address

2. **Complete Your Profile**:
   - Add your full name
   - Upload a profile picture
   - Set your timezone and notification preferences
   - Choose your role (Developer, Designer, QA, Manager, etc.)

3. **Join or Create a Workspace**:
   - Accept an invitation to join an existing workspace
   - Or create a new workspace for your team

### First-Time Setup

**Step 1: Workspace Configuration**
- Set your workspace name and description
- Upload a workspace logo
- Invite team members via email

**Step 2: Customize Your Dashboard**
- Arrange widgets to show the information that matters most
- Set your default view (List, Board, Calendar, or Timeline)
- Configure quick actions for common tasks

**Step 3: Connect Integrations** (Optional)
- Link your GitHub/GitLab account
- Connect Slack for real-time notifications
- Integrate with Jira or other project management tools

## Managing Projects

### Creating a New Project

1. Click the **"+ New Project"** button
2. Fill in the project details:
   - **Project Name**: Choose a clear, descriptive name
   - **Description**: Summarize the project goals
   - **Start Date** and **End Date**
   - **Priority Level**: Low, Medium, High, or Critical
   - **Project Type**: Software Development, Design, Marketing, etc.

3. Add team members:
   - Search for users by name or email
   - Assign roles (Owner, Admin, Member, Viewer)

4. Click **"Create Project"**

### Project Views

CodexFlow offers multiple ways to visualize your projects:

#### **List View**
- Simple, straightforward list of all tasks
- Easy sorting and filtering
- Best for: Quick overview and task management

#### **Board View (Kanban)**
- Visual columns for different task statuses
- Drag-and-drop task movement
- Best for: Agile workflows and sprint planning

#### **Calendar View**
- Tasks displayed by due date
- Month, week, or day view options
- Best for: Deadline management and scheduling

#### **Timeline View (Gantt)**
- Visual timeline of project phases
- Dependencies and milestones
- Best for: Complex projects with multiple phases

#### **Analytics View**
- Charts and graphs showing project health
- Velocity trends
- Resource utilization
- Best for: Performance monitoring and reporting

### Project Settings

Access project settings by clicking the **gear icon** on any project:

- **General**: Update project name, description, dates
- **Team**: Add/remove members, change permissions
- **Workflow**: Customize task statuses and transitions
- **Integrations**: Connect external tools
- **Automation**: Set up rules and triggers
- **Notifications**: Configure alert preferences
- **Archive/Delete**: Close or remove the project

## Working with Tasks

### Creating Tasks

**Quick Create**:
1. Click **"+ Add Task"** in any project
2. Enter task title and press Enter

**Detailed Create**:
1. Click **"+ New Task"** for the full form
2. Fill in details:
   - **Title**: Clear, action-oriented description
   - **Description**: Additional context and requirements
   - **Assignee**: Who will work on this task
   - **Reporter**: Who created the task (auto-filled)
   - **Priority**: Low, Medium, High, Critical
   - **Status**: To Do, In Progress, Review, Done
   - **Due Date**: When the task should be completed
   - **Estimated Hours**: Time estimate for completion
   - **Labels**: Categorize with tags (e.g., bug, feature, enhancement)
   - **Dependencies**: Link to blocking tasks

3. **Attach Files**:
   - Upload documents, images, or design files
   - Link to external resources

4. Click **"Create Task"**

### Task Details Page

Each task has a dedicated page with:

#### **Overview Section**
- Full task description
- Current status and priority
- Assignee and reporter
- Dates and time tracking

#### **Comments Section**
- Team discussion thread
- @ mentions to notify specific people
- Markdown support for formatting
- Emoji reactions

#### **Activity Log**
- Complete history of changes
- Who made each change and when
- Status transitions
- Field updates

#### **Subtasks**
- Break down complex tasks
- Track progress on individual components
- Assign subtasks to different people

#### **Attachments**
- View all uploaded files
- Download or preview inline
- Version history for files

#### **Related Items**
- Linked tasks
- Dependencies (blocks/blocked by)
- Related PRDs or documents

### Task Actions

**Status Updates**:
- Change status with one click
- Or drag tasks between columns in Board view

**Reassignment**:
- Click the assignee avatar
- Select a new team member
- Add a comment explaining the handoff

**Time Tracking**:
- Log time spent on tasks
- Start/stop timer for real-time tracking
- View time reports

**Task Comments**:
- Add updates and ask questions
- Use @ to mention team members
- Upload images inline
- Use markdown for code snippets

## Integrations

### GitHub / GitLab Integration

**Setup**:
1. Go to **Settings ‚Üí Integrations**
2. Click **"Connect GitHub"** or **"Connect GitLab"**
3. Authorize CodexFlow to access your repositories
4. Select repositories to sync

**Features**:
- **Automatic Task Creation**: Create tasks from issues
- **Commit Linking**: Reference tasks in commit messages
  ```
  git commit -m "Fix bug in login #TASK-123"
  ```
- **PR Tracking**: See pull request status in tasks
- **Branch Linking**: Associate branches with tasks
- **Deployment Tracking**: Monitor deployment status

### Slack Integration

**Setup**:
1. Go to **Settings ‚Üí Integrations**
2. Click **"Add to Slack"**
3. Choose your Slack workspace
4. Select a channel for notifications

**Commands**:
```
/codexflow tasks               # View your tasks
/codexflow create [task name]  # Quick task creation
/codexflow status [project]    # Project status
/codexflow help                # Show all commands
```

**Notifications**:
- Task assignments
- Status changes
- @mentions
- Project milestones
- Daily digests

### Jira Integration

**Setup**:
1. Go to **Settings ‚Üí Integrations**

**Features**:

## FAQs

### General Questions

**Q: How many projects can I create?**
A: Depends on your plan:
- Starter: 5 projects
- Growth: 25 projects
- Enterprise: Unlimited

**Q: Can I use CodexFlow offline?**
A: Limited offline access is available. You can view cached data, but changes sync when you're back online.

**Q: How do I invite team members?**
A: Go to Settings ‚Üí Team ‚Üí Invite Members. Enter their email addresses and assign roles.

**Q: Is my data secure?**
A: Yes! We use bank-level encryption (AES-256), secure authentication (OAuth 2.0), and comply with GDPR and SOC 2 standards.

### Task Management

**Q: Can I create recurring tasks?**
A: Yes! When creating a task, enable "Recurring" and set the frequency (daily, weekly, monthly).

**Q: How do I bulk-edit tasks?**
A: Select multiple tasks (checkbox on left), then use the bulk actions menu to update status, assignee, priority, etc.

**Q: Can I import tasks from other tools?**
A: Yes! Use the Import feature (Settings ‚Üí Import/Export) to import from CSV, Jira, Trello, or Asana.

### AI Features

**Q: How accurate is the AI risk analysis?**
A: Our AI models are trained on thousands of projects and achieve 85%+ accuracy. However, always use human judgment for critical decisions.

**Q: Can I customize AI suggestions?**
A: Yes! Go to Settings ‚Üí AI Preferences to adjust sensitivity, focus areas, and notification frequency.

**Q: Does AI learn from my team's data?**
A: Yes, the AI improves over time based on your team's patterns, but your data remains private and isn't shared with other organizations.

### Billing

**Q: Can I change my plan anytime?**
A: Yes! Upgrade or downgrade anytime. Changes are prorated.

**Q: What payment methods do you accept?**
A: Credit/debit cards, PayPal, wire transfer (Enterprise), and invoicing (annual plans).

**Q: Is there a free trial?**
A: Yes! 14-day free trial, no credit card required.

## Troubleshooting

### Common Issues

#### **Can't Log In**

**Solutions**:
1. Check your email and password
2. Try "Forgot Password" to reset
3. Clear browser cache and cookies
4. Try a different browser
5. Contact support if issue persists

#### **Notifications Not Working**

**Solutions**:
1. Check notification settings (Profile ‚Üí Notifications)
2. Verify email isn't in spam folder
3. Check browser notification permissions
4. Confirm Slack integration is active (if using Slack)

#### **Slow Performance**

**Solutions**:
1. Refresh the page
2. Clear browser cache
3. Check your internet connection
4. Try disabling browser extensions
5. Use a modern browser (Chrome, Firefox, Safari, Edge)

#### **Integration Not Syncing**

**Solutions**:
1. Reconnect the integration (Settings ‚Üí Integrations)
2. Check API token validity
3. Verify permissions in the external tool
4. Review sync logs for errors
5. Contact support with error details

#### **Can't Upload Files**

**Solutions**:
1. Check file size (max 50MB per file)
2. Verify file type is supported
3. Check your storage quota
4. Try a different file format
5. Compress large files before uploading
