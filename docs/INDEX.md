## üìä Technology Stack Reference

### Frontend
- **Framework**: Next.js 14 (React 18)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn/ui
- **State Management**: Zustand + TanStack Query
- **Testing**: Jest, React Testing Library, Playwright

### Backend
- **Runtime**: Node.js 18
- **Framework**: Express (in Lambda)
- **Language**: TypeScript
- **Serverless**: AWS Lambda
- **API**: API Gateway

### Database
- **NoSQL**: Amazon DynamoDB
- **Document Store**: Amazon DocumentDB (MongoDB-compatible)
- **Cache**: Amazon ElastiCache (Redis)

### AI/ML
- **Training**: Amazon SageMaker
- **NLP**: Amazon Comprehend
- **LLM**: Amazon Bedrock (Claude, GPT-4)
- **Custom ML**: Python + scikit-learn + TensorFlow

### Infrastructure
- **Cloud**: AWS
- **IaC**: Terraform / AWS CDK
- **CI/CD**: AWS CodePipeline / GitHub Actions
- **Monitoring**: CloudWatch, X-Ray

### Integrations
- GitHub, GitLab, Jira, Slack, Notion, Figma
